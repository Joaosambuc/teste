Explicação:

Este site foi criado com a finalidade única de exercitar as habilidades de programação em html e css do criador.

No menu superior será possivel entrar em diferentes partes do site. A primeira delas fará com que você retorne a esta página. A segunda te levará para uma coletânia de fotos de algumas das aeronaves favoritas do magnífico criador. Ao cliclar na foto, uma página explicando um pouco sobre cada avião se abrirá (genial não?), então você poderá entender o porque do favoritismo.

Na aba contatos você poderá conhecer algumas das redes sociais e formas de contactar (óbvio) o grandioso. Atenção, para comuniar-se com ele será necessario usar uma linguagem que esteja a sua altura. Não tente utitlizar o idioma comum pois isto despertará a fúria do altivo.
